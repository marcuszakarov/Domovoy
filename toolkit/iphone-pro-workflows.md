# iPhone 17 Pro — Real Estate Workflows

## Camera Settings for Property Photography

### Hero Shots (Exterior + Key Rooms)
```
Format:     48MP ProRAW (Settings → Camera → Formats → Apple ProRAW)
Lens:       Main 48MP (1x)
HDR:        ON (Smart HDR handles Thai harsh sunlight)
Grid:       ON (level your shots — crooked walls kill listings)
Macro:      AUTO (for detail shots of finishes, tiles, hardware)
Flash:      OFF always (natural light or bring LED panels)
```

### Camera Control Button Setup
The Camera Control button on the right side is your best friend:
1. **Settings → Camera → Camera Control**
2. Set light press to toggle between: **Zoom** (for switching 0.5x/1x/2x/5x)
3. Set deep press to: **Take Photo**
4. During video: light press to start/stop recording

### Shot List Per Property (Use This Every Time)

**Exterior (5-8 shots)**
- [ ] Front facade, straight-on, eye level
- [ ] Gate/entrance approach
- [ ] Garden/landscaping overview
- [ ] Pool (if applicable) — shoot toward the house, not away
- [ ] Parking/garage
- [ ] Street view showing neighborhood
- [ ] Any view (ocean, mountain, sunset direction)

**Interior — Per Room (3-4 shots each)**
- [ ] Wide shot from doorway (0.5x ultra-wide)
- [ ] Feature shot (kitchen island, bathtub, built-ins)
- [ ] Window/view from inside
- [ ] Detail shot of finishes (countertops, fixtures, flooring)

**Must-Capture**
- [ ] Water heater brand/condition
- [ ] Electrical panel
- [ ] Air conditioning units (brand, age)
- [ ] Water pressure test (run taps, video it)
- [ ] Ceiling for water stains
- [ ] Any damage or wear (document for negotiation)

### 5x Optical Zoom — Unique Advantage
Most agents can't get these shots without a ladder:
- **Roof condition** from ground level
- **Gutter condition** and drainage
- **Upper floor window seals** for water damage signs
- **Ceiling details** (crown molding, light fixtures, cracks)
- **High wall finishes** and paint condition
- **Tree canopy** health near the property

## LiDAR Scanner — Floor Plans & Measurements

### Best Apps for Property Scanning

**RoomPlan (Apple's built-in API)**
- Free, native, fast
- Creates clean 3D floor plans
- Export as USDZ (3D) or screenshot for listings
- Best for: Quick room dimensions during viewings

**Polycam**
- Professional-grade 3D scans
- Export floor plans as PDF, DWG, or OBJ
- Texture capture (shows actual materials)
- Best for: Full property documentation, sending to architects

**Canvas (by Occipital)**
- Schematic-quality floor plans
- CAD-ready exports
- Room-by-room scanning with measurements
- Best for: Renovation planning with clients

### LiDAR Scanning Tips
1. **Walk slowly** — rushing creates gaps in the mesh
2. **Scan room by room**, not the whole property at once
3. **Start from the doorway** of each room
4. **Ensure good lighting** — LiDAR works in dark, but texture needs light
5. **Include ceiling height** — important for Thai properties (many have high ceilings)
6. **Scan outdoor covered areas** — Thai salas, terraces, carports
7. **Total property scan time**: ~15-20 mins for a 3-bed villa

### Quick Measurement Mode
Don't need a full scan? Use the **Measure app** (pre-installed):
1. Open Measure app
2. Point at any surface — it auto-detects edges
3. Tap to set start point, drag to end point
4. Screenshot the measurement
5. Works for: room dimensions, furniture fit checks, window sizes

## Spatial Video — Remote Client Viewings

### When to Use Spatial Video
- International families who can't visit yet (your core Domovoy audience)
- Premium properties where immersion sells
- Properties competing with others — spatial video is a differentiator

### How to Shoot
1. **Hold iPhone horizontally** (landscape) — spatial video requires landscape
2. **Move slowly** through spaces, no quick pans
3. **Pause for 3-5 seconds** in each key area
4. **Narrate** as you go — "This is the master bedroom, facing east for morning light"
5. **Film at eye-level** — this is what living there feels like
6. Clients view on Apple Vision Pro or in 3D on their iPhones

### Spatial Video Walkthrough Script
```
1. Start at the street → walk to entrance (set the neighborhood feel)
2. Pause at front door → open and enter
3. Living area → slow pan, pause on key features
4. Kitchen → show counter space, appliances, view from kitchen
5. Each bedroom → enter from door, show closets, bathroom if en-suite
6. Master bedroom → extra time, show the view
7. Outdoor area → pool, garden, terrace
8. End with the best view the property offers
```

## ProRes Video — Property Tour Videos

### Settings
```
Resolution:  4K at 30fps (smooth, cinematic)
Format:      ProRes (or HEVC if storage is tight)
Stabilization: Action mode OFF (too much crop), use a gimbal or steady hands
Audio:       Built-in mic is fine for narration, use AirPods for wind reduction
```

### Storage Warning
- ProRes 4K = ~6GB per minute
- A 10-minute property tour = ~60GB
- **Solution**: Carry a USB-C SSD (Samsung T7 Shield is waterproof — good for Thailand)
- Transfer immediately after each viewing: Settings → transfer to SSD via Files app

## Photo Pipeline — From Capture to Listing

### Automated with Shortcuts (see shortcuts-guide.md)

```
Capture (iPhone camera)
    ↓
Auto-organize by date + property name (Shortcuts)
    ↓
Select best 15-20 shots (manual)
    ↓
Quick edit in Photos or Lightroom Mobile
    - Straighten horizons
    - Boost shadows (Thai interiors can be dark)
    - Correct white balance (Thai lighting tends warm/yellow)
    - Crop for consistency
    ↓
Export at web resolution (2000px wide, 80% quality JPEG)
    ↓
Add to property listing in GitHub
    ↓
Push → site updates live
```

### Photo Editing Quick Fixes (Photos App)
For each property photo, do this 30-second edit:
1. **Auto enhance** (magic wand icon) — gets you 80% there
2. **Straighten** — use the grid overlay, align to walls/horizon
3. **Shadows +30** — brightens dark Thai interior corners
4. **Warmth -10** — counters yellowish Thai indoor lighting
5. **Crop** to consistent aspect ratio (4:3 for listings)

## Action Button Configuration

Set your Action Button for instant property capture:

**Settings → Action Button → Shortcut → "Property Quick Capture"**

This Shortcut (build it in Shortcuts app):
1. Opens Camera in 48MP ProRAW mode
2. After photos are taken, prompts: "Property name?"
3. Creates a folder with that name in Files
4. Moves today's photos there
5. Optionally starts a voice memo for notes

## Focus Mode — "Property Viewing"

**Settings → Focus → + Custom → "Property Viewing"**

### Allow notifications from:
- Client contacts only
- Line app
- WhatsApp
- Phone calls (repeated calls only)

### Silence everything else:
- Email, social media, news
- Non-essential apps

### Auto-activate:
- When you open the Camera app
- When you arrive at a saved property location
- Between 9 AM - 6 PM on viewing days (set in Calendar)

### Lock Screen:
- Show: Camera, Measure, Polycam, Line
- Widget: Next Calendar event (next viewing)
