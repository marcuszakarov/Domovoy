# Domovoy Field Toolkit

Your daily operations stack for luxury real estate in Phuket.
Hardware: **iPhone 17 Pro** | **Ray-Ban Meta 2** | **GitHub Mobile**

---

## Quick Start — Daily Workflow

### Morning Routine
1. **Ray-Ban Meta 2** on, connected to iPhone via Meta View app
2. Open **Shortcuts** → run "Daily Property Brief" (pulls new listings, weather, exchange rates)
3. Check GitHub Issues for any site update requests
4. Review today's viewings in Calendar

### Property Viewing
1. **Before arrival**: Run "Pre-Viewing Prep" Shortcut → pulls property file, sets camera to ProRAW
2. **On-site with Ray-Ban Meta 2**: Hands-free video walkthrough + AI voice notes
3. **iPhone 17 Pro camera**: Hero shots in 48MP, measurements with LiDAR, spatial video for remote clients
4. **Post-viewing**: Run "Property Wrap-Up" Shortcut → organizes media, drafts listing, pushes to GitHub

### Client Communication
1. **Line** for Thai clients (it's the #1 app in Thailand, non-negotiable)
2. **WhatsApp** for international families
3. **Email** for schools and formal communication
4. Share property cards via **NFC tap** or AirDrop

## Pro Tips

1. **Always carry a portable charger** — LiDAR scans and 4K video drain battery fast
2. **Download Thai offline maps** — Cell coverage in rural Phuket zones is spotty
3. **Keep Ray-Ban Meta charged to 80%+** before viewings — battery life is ~4 hours
4. **Set iPhone Action Button** to open camera in ProRAW mode for instant property shots
5. **Use Focus modes** — Create a "Property Viewing" Focus that silences everything except client calls
6. **Back up to iCloud** same day — Thai humidity + salt air can damage devices unexpectedly
7. **Use AirDrop** to share photos to clients immediately during viewings — huge trust builder
8. **NFC business cards** — Program NFC tags with your contact/WhatsApp link, tap to share
